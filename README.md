# 学习资源管理器 (Learning Resource Manager)

一个纯前端的学习资源管理和任务安排工具，无需后端服务器，数据存储在浏览器本地。

## 📋 项目概述

学习资源管理器是一个功能完整的个人学习管理工具，帮助用户高效组织学习资源（视频、文档、文件）并安排学习任务。采用直观的看板式界面，支持拖放操作和实时数据同步。

**核心特点：**
- 🚀 **纯前端运行**：无需安装依赖，直接打开HTML文件即可使用
- 💾 **本地存储**：使用LocalStorage持久化数据，保护隐私
- 🎨 **现代界面**：响应式设计，适配桌面和移动设备
- 🔄 **数据导入/导出**：支持JSON格式数据备份和恢复
- 🛠️ **模块化架构**：清晰的前后端分离设计

## ✨ 主要功能

### 📚 资源管理
- **分类管理**：支持视频、文档、文件三种资源类型
- **快速搜索**：实时搜索资源标题和描述
- **标签系统**：为资源添加自定义标签
- **批量操作**：支持批量导入和导出资源

### 📋 任务看板
- **看板式布局**：待办、进行中、已完成三列看板
- **拖放操作**：直观的拖放调整任务状态
- **优先级设置**：高、中、低三种优先级
- **状态跟踪**：实时更新任务进度
- **智能排序**：已完成任务自动置顶，支持折叠/展开

### 🔄 数据管理
- **本地存储**：自动保存所有更改，防抖优化性能
- **导入功能**：从JSON文件导入资源和任务
- **导出功能**：导出当前数据为JSON文件
- **数据验证**：确保导入数据的完整性和格式正确

### 🎯 用户体验
- **自定义弹窗**：统一的模态框组件，支持输入、选择、多行文本
- **实时反馈**：操作成功/失败提示
- **响应式设计**：适配不同屏幕尺寸
- **键盘快捷键**：支持快速操作
- **删除确认**：拖动到垃圾桶区域删除任务

## 🛠️ 技术栈

### 前端技术
- **HTML5**：语义化标签和现代API

- **浏览器开发者工具**：调试和测试

## 🚀 快速开始

### 方法一：直接运行
1. 克隆或下载项目到本地
2. 使用浏览器打开 `index.html` 文件
3. 无需任何安装或配置，立即开始使用

### 方法二：本地服务器运行（可选）
```bash
# 使用Python启动HTTP服务器
python -m http.server 8000

# 使用Node.js http-server（需先安装）
npx http-server
```
访问 http://localhost:8000 查看应用

### 首次使用指南
1. **添加资源**：点击"添加资源"按钮，填写资源信息
2. **创建任务**：点击"新任务"按钮，或从资源拖动生成任务
3. **管理任务**：在看板中拖放任务调整状态
4. **导入数据**：点击"导入"按钮导入已有数据
5. **导出备份**：定期导出数据备份到本地
